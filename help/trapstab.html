Traps Tab

The properties of each trap are exposed in a spreadsheet. Each row corresponds to a trap, and each column to a property. Every trap has the following properties:
  • x: the X position of the trap [pixels]
  • y: the Y position of the trap [pixels]
  • z: the Z position of the trap [pixels]
  • A: the relative amplitude of the trap
  • ϕ: the relative phase of the trap [radians]
These values update in real time as you manipulate the traps and can be edited directly by clicking on a cell. Additional columns are added automatically to control properties of structured traps. The available structured trap types are:
  • Tweezer: a single focused spot.
  • Vortex: a Laguerre-Gaussian beam with a topological charge.
  • Ring trap: an annular optical trap.
  • Trap array: a rectangular grid of tweezers, with optional position jitter (fuzz) and a mask to suppress individual sites. Extra properties: nx (columns), ny (rows), separation (spacing in pixels), fuzz (random displacement std dev in pixels).
  • Letter array: a single dot-matrix character (A–Z, a–z, 0–9) rendered as tweezers on a 5×7 grid. Extra property: separation (dot spacing in pixels).
  • Text array: a string of letter arrays laid out side by side. Extra properties: text (the string), separation (dot spacing in pixels).
The extra property cells have tooltips that explain their function.
