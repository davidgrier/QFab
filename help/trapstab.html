Traps Tab

The properties of each trap are exposed in a spreadsheet. Each row corresponds to a trap, and each column to a property. Every trap has the following properties:
  • x: the X position of the trap [pixels]
  • y: the Y position of the trap [pixels]
  • z: the Z position of the trap [pixels]
  • A: the relative amplitude of the trap
  • Ï•: the relative phase of the trap [radians]
These values update in real time as you manipulate the traps and can be edited directly by clicking on a cell. Additional columns are added automatically to control properties of structured traps. These extra columns are not labeled. Instead, the relevant cells have tooltips that explain their function.
